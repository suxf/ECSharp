详细描述

时间流

继承此类可以实现Update实时更新功能

为了方便类的部分初始和性能节省需手动调用 Start(); 函数

每次Update是先执行函数体内容再睡眠等待，所以如果需要精确的时间间隔应当先判定时间再累加时间

◆ SetHighPrecisionMode()

static void ES.Common.Time.TimeFlow.SetHighPrecisionMode ( bool  state)
static

设置高精度模式

在某些情况的服务中需要毫秒级无误差支持可打开此项

在机器设备支持的情况下可支持毫秒无误差更新，具体视执行函数体内容而定

此模式会占用更多的CPU资源，低精度不占用CPU资源，但会存在毫秒误差，不过结果趋向是相同的

默认是低精度模式，可在任何时间变更精度模式，要注意精度模式是影响整个框架的所有更新模式

参数
state打开状态
