◆ HeartCheckPeriod

int ECSharp.Network.Sockets.HyperSocket.HyperSocketConfig.HeartCheckPeriod = 4000

[S] 服务端心跳检测超时周期 单位 毫秒ms

系统会根据此时间来接受心跳检测

◆ HeartSendPeriod

int ECSharp.Network.Sockets.HyperSocket.HyperSocketConfig.HeartSendPeriod = 10000

[C] 客户端心跳发送周期 单位 毫秒ms

系统会根据此时间来循环发送心跳用来检测

◆ HeartTimeOut

int ECSharp.Network.Sockets.HyperSocket.HyperSocketConfig.HeartTimeOut = 10000

[C+S] 心跳超时时间 单位 毫秒ms

超过心跳服务器/客户端自动断开释放

可以通过调整心跳检测周期来更改误差

◆ SSLMode

int ECSharp.Network.Sockets.HyperSocket.HyperSocketConfig.SSLMode = 0

[S] 使用安全传输模式

传输可以决定在安全协议下 哪种通信使用加密传输 默认TCP/UDP都是用加密传输

0 TCP/UDP全使用加密传输 1 TCP使用加密传输 2 UDP使用加密传输
