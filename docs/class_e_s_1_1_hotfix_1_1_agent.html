热更新代理

只有继承这个代理类才能在热更层使用其泛型的变量值

如果继承AbstractAgent，则需要用IAgent来获取

继承此类使用数据层变量请用 self 代替 this

代理类中函数如果存在委托情况，请注意如仅存于非热更层的委托可能存在问题

此类是代理类，请不要使用代理类来直接声明变量或属性

当然如果真的有需要，可以使用 AgentDataPivot 类来存储变量实现热更层声明变量

或者使用特性[CopyAgentValue]来持有代理类的成员变量

热更层可以继承时间流接口，每次热更重载后时间流重置

不要在代理类使用构造函数，请使用Initialize来初始化一些内容

当存在对应关系的代理 AgentData 继承对象时，会在构建代理数据 AgentData 时自动构建此类继承的子类

当然如果不要自动构建可以在 AgentData 继承类添加 [NotCreateAgent] 特性, 即可手动通过 GetAgent 函数创建

如果确实需要在热更层新建字段或属性，并且希望可以重载之后值也可以保留，那么可以使用 [CopyAgentValue] 特性
