热更新代理数据

只有通过继承代理数据的类才能使用热更代理层函数

此类事代理数据类，代理类的变量或属性需要从此声明来使用

需要在热更层使用的变量或者属性需要使用 public 访问修饰符

当存在对应关系的代理 Agent 继承对象时，会在构建代理数据 AgentData 时自动构建 Agent 继承对象

当然如果不要自动构建可以在 AgentData 继承类添加 [NotCreateAgent] 特性, 即可手动通过 GetAgent 函数创建

如果确实需要在热更层新建字段或属性，并且希望可以重载之后值也可以保留，那么可以使用 [CopyAgentValue] 特性
