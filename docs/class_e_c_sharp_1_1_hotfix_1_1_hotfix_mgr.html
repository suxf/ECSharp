载入热更新模块

读取不能保证所有情况都是正常的，建议用try-catch捕获异常，以确保未正确替换，还能保持旧环境继续运行

默认主入口包含一个静态Main函数 public static void Main(string[] args){}

参数
assemblyFileName程序集文件名(后缀小写或不写且程序集需要在运行根目录下)
classFullName含有 public static void Main(string[] args){} 程序集下热更模块主入口类全称(即命名空间和类名)
args传入热更层字符串数组
entryMethodName入口函数名称，默认不指定为 Main
返回
本次加载是否执行，进入执行且完成为true，未执行为false
